# Исправление ошибки 404

## Проблема
Приложение показывает 404 ошибку для всех страниц.

## Решение

Проверь следующее:

1. **Перезапусти dev server** после изменений в `next.config.js`:
   ```bash
   # Останови сервер (Ctrl+C)
   # Затем запусти снова:
   npm run dev
   ```

2. **Проверь, что файлы существуют**:
   - `app/page.tsx` - главная страница
   - `app/trainer/page.tsx` - тренажер
   - `app/profile/page.tsx` - профиль
   - `messages/ru.json` - файл переводов

3. **Очисти кеш Next.js**:
   ```bash
   rm -rf .next
   npm run dev
   ```

4. **Проверь консоль браузера** на наличие ошибок

5. **Проверь консоль терминала** где запущен `npm run dev`

## Если проблема сохраняется

Временно отключи middleware для проверки:

1. Переименуй `middleware.ts` в `middleware.ts.bak`
2. Перезапусти сервер
3. Если страницы работают, проблема в middleware
4. Верни middleware и проверь конфигурацию
